# **App Name**: Feline & Friend

## Core Features:

- Product Browsing and Discovery: Allow customers to browse cats and pet products with filtering, search, and detailed product views.
- Shopping Cart and Checkout: Enable customers to add items to a cart, manage quantities, and complete purchases using a dynamically generated delivery form.
- User Authentication and Roles: Implement Firebase Authentication for customer and admin roles, managing access and profile information.
- Admin Product Management: Provide an admin interface to create, edit, and manage products, including uploading images and setting details.
- Dynamic Delivery Form Builder: Allow admins to create and modify the delivery form schema, influencing the customer checkout experience.
- Order Management: Enable admins to view, filter, and update orders, with detailed customer and delivery information.
- AI Product Description Tool: The admin has access to a tool that uses AI to suggest a better product description for a given product.

## Style Guidelines:

- Primary color: Light and soft lavender (#E6E6FA) for a calming and friendly feel.
- Background color: Very light gray (#F5F5F5), almost white, creating a clean and modern look.
- Accent color: Pale pink (#FFB6C1), for interactive elements and highlights, to complement the lavender.
- Body and headline font: 'PT Sans' (sans-serif) for a modern yet accessible design.
- Use a set of simple, line-based icons with rounded corners, reflecting the pet-friendly theme.
- Mobile-first design with a Daraz-style bottom navigation for key customer sections. Use multi-column layouts on desktop for product grids.
- Subtle, gentle animations for transitions and feedback, avoiding harsh or jarring movements.
